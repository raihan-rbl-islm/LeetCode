# 345. Reverse Vowels of a String

**Difficulty:** Easy  
**Topic:** Two Pointers, String  
**Link:** [LeetCode 345](https://leetcode.com/problems/reverse-vowels-of-a-string/)

## Intuition

We can solve this efficiently using the **Two Pointer** technique. 
1. Initialize one pointer `i` at the start and `j` at the end of the string.
2. Move `i` forward until it finds a vowel.
3. Move `j` backward until it finds a vowel.
4. Swap the characters at `i` and `j`, then move both pointers inward.
5. Repeat until the pointers meet.

This ensures we only reverse the vowels while keeping the rest of the string intact.

## Complexity

* **Time Complexity:** $O(N)$ — We traverse the string exactly once (each character is visited by either `i` or `j`).
* **Space Complexity:** $O(1)$ — We perform the swap in-place using only constant extra space.

## Code (C++)

```cpp
class Solution {
public:
    // Helper function to check if a character is NOT a vowel.
    // 'inline' suggests the compiler to embed this code directly to save function call overhead.
    inline bool notVowel(char c) {
        return !(c == 'a' || c == 'e' || c == 'i' || c == 'o' || c == 'u' || 
               c == 'A' || c == 'E' || c == 'I' || c == 'O' || c == 'U');
    }

    string reverseVowels(string s) {
        int i = 0, j = s.size() - 1;
        
        while (i < j) {
            // Move left pointer forward until a vowel is found
            while (notVowel(s[i]) && i < j) i ++;
            
            // Move right pointer backward until a vowel is found
            while (notVowel(s[j]) && i < j) j --;
            
            // Swap the vowels
            swap(s[i], s[j]);
            
            // Move pointers inward after swap
            i ++, j --;
        }
        return s; 
    }
};
